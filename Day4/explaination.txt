Q1. HTTP server kya hota hai?
Ans: HTTP server wo hota hai jo request get karta hai from client side se and reponse data deta hai server ko.
    for example: agar user browser ke address bar par search karta hai google.com to it means user ne client se server ko request bheji hai ki user ko google.com chahiye. ab server check karega ki google.com kis IP par assesign hai aur kya ye real world m aisa kuch exist karta hai ya nhi agar karta hai to server response kar denge client ko.

Q3. First HTTP server?
Ans: Node.js hume JavaScript ki help se server-side programming karne deta hai. Node ke andar ek built-in HTTP module hota hai jiska use karke hum apna first HTTP server bana sakte hain. 
Filename: server.js
Code Explanation: 
    1. HTTP module import karna, Ye Node.js ka built-in HTTP module hai. Iski help se hum HTTP server create karte hain. Iske liye koi external package install karne ki zarurat nahi hoti.
    2. HTTP server create karna, http.createServer() ek server create karta hai. Ye ek callback function leta hai jisme:
            2.1 req ‚Üí client se aayi request
            2.2 res ‚Üí server ka response
        res.end() ka kaam:
            2.3 Client ko response bhejna
            2.4 Request ko close karna
    3. Port define karna, Port ek number hota hai jaha server listen karta hai. Commonly development me 3000 use hota hai.
    4. Server start karna. server.listen() server ko start karta hai. Ab server is URL par available hoga:
        http://localhost:3000
