Q1. Nodejs actually kya hai?
Ans: Nodejs ek environment hai jisme hum JavaScript ko server par run karte hai.
Later: Js ko sirf browser par hi run kar sakte the.
Now: Js ke through hum backend , API, DB , Auth, etc 

Q2. Nodejs kyo use karte hai?
Ans: Nodejs ko hum isly use karte hai ki kyonki hum JavaScript se server bana sakte hai.
    1. Fast hoti hai => Nodejs jaldi kaam karta hai, isly website slow nhi hoti hai
    2. Single lanague use hoti hai => basically isme frontend and backend dono m hi JavaScript hi use hoti hai
    3. Multiple user access kar sakte hai same time par => lag/ hang nhi hogi website , lightweight rhti hai
    4. Realtime application built => Chat system, live streaming application, etc.
